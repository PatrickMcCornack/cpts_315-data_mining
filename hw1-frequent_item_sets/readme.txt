How to use this program:
-Simply run the script in the same folder as browsing-data.txt
-The script will create a csv-data.txt which is a comma-delimited version of browsing-data.txt
-The script will also automatically output results in results.txt
-If a file other than browsing-data.txt is desired, then change the value of the "unformatted file" variable in the script
