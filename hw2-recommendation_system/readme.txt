The script McCornack_CPTS215_PA2 will create an "output.txt" file that has the top 5 movie recommendation for each user based on comparing similarities of movies. 
Simply run the script to generate the list of movie recommendations. 

The format of the output file is:
user-id-1 movie-id-1 movie-id-2 movie-id-3 movie-id-4 movie-id-5
user-id-2 movie-id-1 movie-id-2 movie-id-3 movie-id-4 movie-id-5
..
